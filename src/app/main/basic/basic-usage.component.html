Basic Usage

Basic configuration of the map. In this example is shown how to define the map component, setting the center and zoom levels.

Base layers

It can be configured which layer or layers will be the base layers of the map. By default, if none is provided it is used 'OpenStreetMap' base layer.

The different base layer/s are taken from the extension 'leaflet-providers' that contains configurations for various free tile providers. Please, for production environments take into account what leaflet-providers developers says:
